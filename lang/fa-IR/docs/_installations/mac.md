### سیستم عامل مک

#### Homebrew

می‌توانید Yarn را از طریق [ مدیر بسته Homebrew ](http://brew.sh/) نصب کنید. با استفاده از این روش چنانچه Node.js بر روی سیستم شما نصب نباشد، آن نیر نصب می‌شود.

```sh
brew update
brew install yarn
```
