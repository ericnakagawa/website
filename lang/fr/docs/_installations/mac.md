### macOS

#### Homebrew

Vous pouvez installer Yarn via le [Gestionnaire de paquet pour mac Homebrew](http://brew.sh/). Ceci va également installer Node.js s’il n’est pas déjà installé.

```sh
brew update
brew install yarn
```
