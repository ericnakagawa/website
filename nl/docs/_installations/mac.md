### macOS

#### Homebrew

Je kunt Yarn installeren met behulp van de [Homebrew package manager](http://brew.sh/). Hiermee zal ook Node.js worden geïnstalleerd indien deze nog niet geïnstalleerd is.

```sh
brew update
brew install yarn
```
